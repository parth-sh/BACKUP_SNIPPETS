# Installation
```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```
[Installation](https://brew.sh/)

## Cleanup
```
brew update && brew upgrade && brew cleanup
```

## Search
```
brew search <programe_name>
brew list
```

## Uninstall package
```
brew uninstall <package>
brew autoremove
```
